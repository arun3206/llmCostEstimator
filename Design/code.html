Financial Calculation Methodology

The Standard Formula

Total Cost = (Input Tokens × Input Rate) + (Output Tokens × Output Rate)
Input Logic

We calculate input tokens based on the CL100K_Base tokenizer for OpenAI models and standard character-to-token ratios for other providers (approx. 4 chars/token).

Output Logic

Outputs are modeled based on three distribution tiers: Short (50 tokens), Medium (300 tokens), and Detailed (1,200 tokens).
